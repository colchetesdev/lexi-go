📱 LexiGo

LexiGo is a React Native application designed to help Brazilian users learn and master English vocabulary focused on technology. Whether you're a developer, student, or tech enthusiast, LexiGo makes it easier to understand and use key technical terms in English with confidence.

🚀 Features
📚 Curated Tech Vocabulary
Learn essential English terms used in programming, software development, and IT.
🇧🇷➡️🇺🇸 Portuguese to English Learning
Clear translations and contextual explanations tailored for Brazilian Portuguese speakers.
🔊 Pronunciation Support
Listen to correct pronunciations to improve your speaking and listening skills.
🧠 Interactive Learning
Quizzes, flashcards, and repetition techniques to reinforce memory.
📈 Progress Tracking
Monitor your learning journey and track improvement over time.
